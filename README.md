# msb_socius_2016

Replication files for Martin, Slez, and Borkenhagen (2016)

## Contents

`msb_2016.Rdata` 

Collection of data objects used for analysis. 

`functions.R`

Custom functions. 

`msb_2016_clean.R`

Code for cleaning data in `msb_2016.RData`. Must be run before analysis.

`msb_2016_analysis.R`

Code for recreating findings.
